/proc/resize_icon Upscales an icon using rust-g. You really shouldn't use this TOO often, as it has to copy the icon to a temporary png file, resize it, fcopy_rsc the resized png, and then create a new /icon from said png. Cache the output where possible.
/proc/get_job_hud_icon Returns the (isolated) security HUD icon for the given job.
